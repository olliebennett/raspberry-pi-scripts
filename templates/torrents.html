Torrents

This Raspberry Pi runs a Deluge-based torrent client.

Accessing Deluge

Deluge runs a Web-based user interface for adding/removing and otherwise controlling the state of torrents in the system. Open the Web UI.

An alternative to the WebUI is to install the Deluge client locally, and connect it to the remote daemon on the Raspberry Pi.

  1. Install Deluge.
  2. Disable "Classic Mode" under "Edit" > "Preferences" > "Interface".
  3. Open the "Edit" > "Connection Manager"
  4. Add a new host with the following parameters:
    Hostname*:
    Port:
    Username:
    Password:

    * The hostname/IP can be replaced by raspberrypi on a local network, assuming Bonjour is installed on your machine.

Using Downloaded Files

Access the completed files by accessing the shared folder as described here. The torrent-related subfolders are:

  • /torrents/downloading - partially downloaded files.
  • /torrents/completed - completed (typically seeding) files.
  • /torrents/watch - a folder monitored by Deluge. Add .torrent files here and they'll be downloaded.
  • /torrents/torrent-backups - a store of all previously-added torrent files.

Other Suggestions

  • Use a tool to add torrents directly from browser to RPi server, such as DelugeSiphon (chrome) or BitTorrent WebUI+ (firefox).
